  • gRPC 架构概览

  • v3 API 参考

这是一个允许 gRPC-Web 客户端和兼容 gRPC 的服务器桥接的过滤器，如下 https://github.com/grpc/grpc/blob/master/doc/PROTOCOL-WEB.md.
