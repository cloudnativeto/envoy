健康检查¶

  • 健康检查过滤器 架构概述

  • v3 API reference

  • 此过滤器通过参数 envoy.filters.http.health_check 来配置。

Note

请注意如果调用了管理端点 /healthcheck/fail ，过滤器将会自动地在健康检查中失败并标示 x-envoy-immediate-health-check-fail 标头。（ /healthcheck/ok 管理端点可以逆反此行为）。
