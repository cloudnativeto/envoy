Lua 过滤器¶

在这个例子中，我们会展示 Lua 过滤器在 Envoy 代理中是如何使用的。 Envoy 代理配置包括一个 Lua 过滤器，此过滤器包含了在 这个 文档中记录的两个函数，即 envoy_on_request(request_handle) 和 envoy_on_response(response_handle) 。

运行沙盒¶

以下文档说明了上述对于 Envoy 容器编译环境进行设置的过程。

步骤 1：安装 Docker¶

确保你已安装较新版本的 docker 和 docker-compose 。

一个最简单的方法来完成此事就是使用 Docker Desktop 。

步骤 2：克隆 Envoy 仓库¶

如果你还没有克隆 Envoy 仓库，用如下方式进行克隆：

SSH

HTTPS

git clone git@github.com:envoyproxy/envoy

git clone https://github.com/envoyproxy/envoy.git

步骤 3：构建沙盒¶

$ pwd
envoy/examples/lua
$ docker-compose pull
$ docker-compose up --build -d
$ docker-compose ps

    Name                     Command               State                            Ports
--------------------------------------------------------------------------------------------------------------------

lua_web_service_1   node ./index.js                  Up      0.0.0.0:8080->80/tcp
